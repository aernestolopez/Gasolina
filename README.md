# ⛽ GasoPrice - Comparador de Combustible en Tiempo Real

Una aplicación web ligera y rápida para encontrar las gasolineras más baratas en España utilizando datos en tiempo real.

## 🚀 Características
- **Datos en Vivo:** Conexión directa con la API de precios de combustible.
- **Filtrado Inteligente:** Busca por municipio, marca de gasolinera y tipo de combustible (Gasolina 95, Diésel, etc.).
- **Optimización de Rendimiento:** Sistema de **caché inteligente** mediante `localStorage` (10 min) para reducir latencia y consumo de datos.
- **Interfaz Responsiva:** Diseño adaptado a móviles y escritorio con indicadores visuales (*Badges*) para las mejores ofertas.

## 🛠️ Tecnologías
- **Frontend:** HTML5, CSS3 (Grid & Flexbox), JavaScript Moderno (ES6+).
- **API:** Integración con servicios REST de precios de carburantes.
- **Lógica:** Algoritmos de filtrado y ordenación dinámica de objetos JSON.
